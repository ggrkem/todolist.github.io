Node js Nasıl Yüklenir?

Öncelikle gerekli olan sürümü bilgisayarımıza yükleyip ardından windows powershell i çalıştırmamız gerekiyor.
Kurulum işlemi tamamlandıktan sonra kurulumun başarılı olup olmadığını node -v veya node --version komutları ile kontrol edebilirsiniz.Linux dağıtımlarında Node.js kurulumunun diğer bir yolu Node.js kaynak kodlarını derlemektir.Kurulum için ilk olarak kaynak kodlarını resmi Node.js sitesinden indirin daha sonra kaynak kodları herhangi bir klasöre çıkartın.Derleme işlemi için gerekli olan gcc ve g++ paketlerini aşağıdaki komutları ile yükleyin .sudo apt-get install gccsudo apt-get install g++Gerekli paketleri indirdikten sonra terminal ile kaynak dosyalarının bulunduğu dizine gelerek sırayla aşağıdaki komutların çalıştırın.  ./configuremakesudo make installKurulum sırasında herhangi bir hata ile karşılaşırsanız işletim sisteminizdeki paketleri tekrar güncelleyerek denemeniz faydalı olacaktır.
